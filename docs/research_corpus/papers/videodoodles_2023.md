# VideoDoodles: Hand-drawn Animations on Videos With Scene-aware Canvases

- **Year**: 2023
- **Authors**: Adobe Research
- **Conference/Source**: SIGGRAPH 2023
- **Topic**: Multimedia & Effects
- **Status**: Visual Innovation

## Core Contribution
Allows users to add hand-drawn animations that stick to objects in the video using scene-aware canvases.

## Relevance to Project
Points towards advanced B-roll/Effects that go beyond just stock footage.
