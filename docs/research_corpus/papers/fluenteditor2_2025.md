# FluentEditor2: Multi-agent LLM Workflow for Script and Subtitle Refinement

- **Year**: 2025
- **Authors**: Wang et al.
- **Conference/Source**: PDF Link: [local_pdf](file:///home/lokesh/ai%20video%20editor/docs/research_corpus/pdfs/fluenteditor2_2025.pdf)

## Core Contribution
Iterative multi-agent editing/review loops for cleaner output. Uses one agent to draft subtitles, another to critique sync/grammar, and a third to fix.

## Relevance to Project
Useful for our `add_subtitles` feature to move from "raw transcript dump" to "polished narrative captions."
